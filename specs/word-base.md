# VATOS Word Design Guide

## Purpose

이 문서는 VATOS `DESIGN.md`의 공통 디자인 기준을 기반으로, Word 또는 일반 문서 산출물에서 페이지 흐름, 표 깨짐, 불필요한 빈 페이지, 긴 본문 가독성 문제를 방지하기 위한 기본 규격이다.

공통 브랜드 컬러, 로고, 타이포그래피, 디자인 톤, Do/Don't, 수정 가능성 원칙은 `DESIGN.md`를 따른다.

## Basic Rules

- Word 또는 일반 문서 산출물 작성 시 이 파일을 우선 참고한다.
- 기본 여백을 기준으로 본문, 표, 그림, 주석이 페이지 안에서 안정적으로 배치되도록 한다.
- 표가 한 페이지에서 깨질 경우 표를 분리하거나 요약 표와 상세 표로 나눈다.
- 표 안에 긴 문장과 다중 bullet을 과도하게 넣지 않는다.
- 불필요한 빈 페이지가 생성되지 않도록 과도한 페이지 나누기, 큰 여백, 빈 문단을 만들지 않는다.
- 제목 계층은 명확하게 유지하고, 본문은 짧은 문단과 목록 중심으로 작성한다.
- 세부 디자인 판단은 `DESIGN.md`의 공통 규칙을 따른다.

## Review Checklist

- [ ] 표가 페이지 폭을 넘어가지 않는가?
- [ ] 표가 중간에서 읽기 어렵게 깨지지 않는가?
- [ ] 불필요한 빈 페이지가 생성되지 않았는가?
- [ ] 제목, 본문, 표, 주석의 계층이 명확한가?
- [ ] VATOS 공통 디자인 기준은 `DESIGN.md`를 따랐는가?
